# CoreStructuresX

CoreStructuresX is a Paper-only modular structure plugin for Minecraft servers.

All official information about the plugin will be available on the CoreStructuresX website:

> **Website:** https://icewolf23x.github.io/CoreStructuresX-website/

The website is intended to contain the complete and up-to-date information about:

- plugin overview
- features
- installation
- configuration
- FAQ
- troubleshooting
- downloads / releases
- platform support
- Paper setup
- WorldEdit / FAWE requirements
- structure pack setup
- schematic marker signs
- JSON persistence and generated instances

This GitHub repository is kept only for issue tracking.

---

## Issue Tracking

Use the GitHub **Issues** section to report problems, request improvements, or track known bugs.

Before opening an issue, please check the official website first:

> https://icewolf23x.github.io/CoreStructuresX-website/

When opening an issue, include as much useful information as possible.

For bug reports, include:

- CoreStructuresX version
- server platform
- Paper version
- Java version
- installed WorldEdit or FAWE version
- relevant pack files, especially `pack.yml`, `structure.yml`, and `groups.yml`
- relevant schematic marker setup, if the issue involves generation
- instance JSON details, if the issue involves persistence or restart behavior
- console errors or stack traces
- clear steps to reproduce the issue
- expected behavior
- actual behavior

For feature requests, include:

- what the feature should do
- why it would be useful
- any examples of the desired behavior
